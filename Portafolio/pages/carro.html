Control de carro con Arduino

Aplicación móvil para controlar un carro electrónico vía Bluetooth
Flutter Arduino Bluetooth IoT

Esta aplicación móvil desarrollada en Flutter permite controlar un carro electrónico construido con Arduino a través de una conexión Bluetooth. La app cuenta con una interfaz intuitiva que incluye una pantalla de selección de dispositivos Bluetooth disponibles y un control remoto con botones direccionales.

El carro está equipado con un módulo Bluetooth HC-05 o HC-06 que recibe los comandos enviados desde la app. Los movimientos (adelante, atrás, izquierda, derecha y parada) se transmiten mediante caracteres simples que el Arduino interpreta para accionar los motores.

Este proyecto combina desarrollo móvil con electrónica y programación de microcontroladores, ideal para aprender sobre IoT y comunicación inalámbrica.

Características principales

  • Conexión Bluetooth con HC-05/HC-06
  • Selección de dispositivo Bluetooth
  • Control remoto con botones direccionales
  • Movimientos: adelante, atrás, izquierda, derecha
  • Botón de parada de emergencia
  • Código Arduino para interpretar comandos
  • Interfaz responsive y minimalista
