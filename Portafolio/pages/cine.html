Base de datos para cine

Diseño, creación y administración de BD en servidor Ubuntu
PostgreSQL Ubuntu Server SQL

Este proyecto consistió en el diseño completo, desde cero, de una base de datos relacional para gestionar todas las operaciones de un sistema de cine, incluyendo películas, funciones, salas, boletos y clientes.

La implementación se llevó a cabo utilizando PostgreSQL, instalado y configurado sobre un entorno de Ubuntu Server. El proceso abarcó desde el modelo entidad-relación y normalización de tablas, hasta la creación de procedimientos almacenados, triggers y vistas para facilitar las consultas y garantizar la integridad de la información.

Además del diseño de la estructura, se administraron los permisos de acceso y se implementaron políticas de seguridad y respaldos, simulando un entorno de producción real donde la disponibilidad y protección de los datos es fundamental.

Características principales

  • Diseño del Modelo Entidad-Relación
  • Normalización de esquemas
  • Creación de procedimientos almacenados y vistas
  • Implementación de triggers para reglas de negocio
  • Configuración de entorno en Ubuntu Server
  • Administración de permisos y usuarios
  • Motor de PostgreSQL
